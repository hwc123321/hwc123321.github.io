研一总结PPT发言

Posted on 2021-12-17

一、迷茫
1.早早毕业答辩，经历了比较长的假期，迷迷糊糊的来到学校，开始研究生生活，开始时的不适应，到慢慢习惯现在的生活，一次偶然的机会了解到课题组，刚到课题组，首先发现与其他课题组不也一样的就是，师兄师姐在哪？后面才知道他们都在外面试验站待着。
二、归属
2.1首先加入课题组就是想着怎么能融入课题组，通过每周一次的仓屋榜取土和日常的小聚餐，我们研一的关系和感情迅速加深。
2.2国庆去枝江试验站协助师兄师姐做实验，入学以来第一次接触师兄师姐，感受到了师兄师姐的温暖。
2.3去和师兄去泉河试验站测水稻水样总氮总磷含量，实验中自己出来很多问题，摔碎试管、倒掉了还没测却已经配好的样品…师兄都没有责怪我，反而让我感到内心十分愧疚。
2.4和华为去兴山出差
三、憧憬
3.1时间过得很快研一上学期转眼就过去了，从开始的迷茫到，慢慢找到自己的目标和归属感，现在更多是对未来两年多时间的憧憬，希望自己不要浪费宝贵的时间，多和老师还有师兄师姐学习，弥补自己在各方面的不足。也希望课题组越来越好！

hexo-github搭建博客一些常见的问题

Posted on 2021-12-16

1.在下载hexo时
1.1 npm install hexo-cli -g
会出现报错情况，1pakceg正在安装

2.在连接github时
2.1找到安装目录下_config.yml文件，用notepade打开修改type及以下的文件为：
type: git
repository: https://github.com/hwc123321/hwc123321.github.io.git（你的连接仓库的地址）
branch: main
2.2连接hexo与github
新版github更新后，在输入hexo d后输入用户名后，提示你输入密码，输入的密码你需要在github开发者设计中，找到Personal access tokens，然后复制密匙，输入到提示输入的密码中，注意密匙使用一次后就会隐藏，下次使用就只能重新创建node！
3.修改主题时
3.1fatal: unable to access ‘https://github.com/hwc123321/hwc123321.github.io.git/': Failed to connect to github.com port 443 after 21273 ms: Timed out网速太慢，重试几次，不然博客界面不显示。
